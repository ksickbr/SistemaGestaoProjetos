# Sistema de Gestão de Projetos

Projeto desenvolvido para a disciplina de Programação de Soluções Computacionais.

## Objetivo

O sistema tem como objetivo auxiliar no gerenciamento de projetos, equipes, usuários e tarefas, permitindo o cadastro, a organização e o acompanhamento das informações principais de cada projeto.

## Funcionalidades

- Cadastro completo de usuários
- Listagem de usuários
- Cadastro de projetos com gerente responsável
- Listagem de projetos
- Cadastro de equipes
- Listagem de equipes
- Associação de usuários a equipes
- Alocação de equipes em projetos
- Cadastro de tarefas vinculadas a usuários e projetos
- Listagem de tarefas
- Relatório geral de acompanhamento

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Programação Orientada a Objetos
- Estrutura baseada no padrão MVC
- Git e GitHub

## Como executar

1. Clonar ou baixar o repositório.
2. Abrir o projeto em uma IDE Java, como IntelliJ IDEA ou Eclipse.
3. Executar o arquivo `Main.java`.
4. Utilizar o menu exibido no console.
